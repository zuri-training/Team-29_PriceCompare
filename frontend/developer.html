Developer Tools

Haggle was built with a library of frontend, backend and design technologies. This section contains the tools and libraries on which Haggle was built with, with a link to the code samples and setup instructions.

Design Library

When building haggle, we designed various symbols that were reused throughout the site, such as typography, colors, and buttons. We converted these symbols into components for easy use by all the design team across our designs.

  • Typography - Font Inter -We used font inter for our design because whether used at large or small sizes, Inter maintains great readability due its tall height. Not only does this make it easier to differentiate between upper and lowercase letters, but it also helps to create space between lines of text.
  • Grid System - We used the 8pt grid to make our designs consistent and scalable.
  • Colour - #2960EE, #EEB729

Frontend Libraries

  • The HTML was used to give structure, and to add and modify contents in the platform. It was also used to create validated forms in the platform, like the login, sign up and contact forms.
  • The CSS was used to give additional styling and layout to the web pages.
  • JavaScript was used to give functionality to the page. It was used to make the page interactive for a good user experience.

Backend Libraries

Django was used as the backend framework for haggle because of its scalability and flexibility for python languuage. Django inbuilt authentication was used to for the user authentication.

Database

Haggle utilizes the Postgresql database to store all its data.

Scraping Tool

Merchants data in haggle were collected using web scraping. Selenium was used for scraping these data.
